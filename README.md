## 🛫 소개
> 스프링도 공부하면서 코틀린도 공부할 방법은 없을까?
>
> 코틀린의 장단점과 자바와의 차이점을 어떻게 이해해야 할까?

- 기존 자바로 구현했던 기능들을 코틀린으로 구현해 보기
- 새로 배운 기술들을 적용해 보기
- 자바와 코틀린의 차이점과 장단점을 직접 구현해서 체감해 보기
- 자바보다 코틀린을 사용했을 때 더 좋은 상황은 무엇인지 알아보기
